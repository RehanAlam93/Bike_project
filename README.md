# 🚲 Bike Price Prediction Project

## 📌 Project Overview

This project is a **Machine Learning Regression Project** that predicts the selling price of a bike based on different features.

The main goal of this project is to build a machine learning model that can estimate the price of a bike using historical bike data.

---

## 🎯 Problem Statement

To develop a Machine Learning model that predicts bike prices based on available features such as bike specifications, ownership details, kilometers driven, and other important factors.

---

## 📂 Dataset Information

The dataset contains information about used bikes and their selling prices.

### Features:
- Bike Name
- Brand
- Year
- Selling Price
- Present Price
- Kilometers Driven
- Owner Details
- Fuel Type
- Seller Type
- Transmission Type

## 🧹 Data Preprocessing

Performed following preprocessing steps:

- Removed unnecessary columns
- Checked missing values
- Converted categorical features into numerical form
- Applied encoding techniques
- Prepared data for Machine Learning models

---

## 📊 Exploratory Data Analysis

Visualizations were created to understand:

- Price distribution
- Relationship between features
- Feature importance
- Correlation between variables

---

## 🤖 Machine Learning Models Used

Different regression algorithms were tested:

- Linear Regression
- Decision Tree Regressor
- Random Forest Regressor

---

## 📈 Model Evaluation

Models were evaluated using:

- R² Score
- Mean Absolute Error (MAE)
- Mean Squared Error (MSE)

---

## 🚀 Project Output

The final trained model can predict the approximate selling price of a bike based on input features.
